== Description ==

Couponer is a basic coupon plugin that allows you to easily create coupons that can be printed by your customers.

<a href=\"http://couponer.zyphon-media.net/demo/\" title=\"Demonstration Page\">VIEW DEMO</a>

<a href=\"http://couponer.zyphon-media.net/\" title=\"Project Homepage\">Project Homepage</a>


== Installation ==

1. Upload the directory and it's contents to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To place in page template: Insert `<?php if (function_exists('display_coupons')) { display_coupons(); } ?>` ; To place with shortcode: Insert `[display_coupons]`
4. Replace the url of your logo that appears in the coupon footer if desired.
5. Create some coupons.
